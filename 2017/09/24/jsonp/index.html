一、开始

首先我们要明确 JSON 和 JSONP 的区别

  • JSON：JSON 是一种基于文本的数据交换方式，或者叫做数据描述格式
  • JSONP：是 JSON 的一种“使用模式”，可用于解决主流浏览器的跨域数据访问的问题
    两者不是同一个概念，JSONP 就是利用 src 属性的跨域能力来实现跨域通讯。
